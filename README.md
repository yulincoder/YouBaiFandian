# YouBaiFandian
### 有百翻典()

####	顾名思义，一个抓取有道词典和百度翻译写的翻译和查单词小工具。
Functions:
1. 单词查询
2. 句子翻译
3. 本地存储查询历史
4. 按照单词查询过的频率进行历史记录排序
5. 历史记录展示项选择
